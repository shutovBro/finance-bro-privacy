# Privacy Policy — Finance Bro

_Last updated: 2026-06-16_

Finance Bro ("the app") is an offline personal finance tracker. Your privacy
is a core design goal of the app.

## Data we collect

**None.** Finance Bro does not collect, transmit, sell, or share any personal
data. The app has no user accounts, no analytics, no advertising, and no
tracking of any kind.

## Where your data lives

All data you enter — accounts, transactions, budgets, categories — is stored
**only on your device**, in a local database. It never leaves your device
except when *you* explicitly export a CSV backup to a location you choose.

## Network access

The app makes a single type of network request: fetching public currency
exchange rates from `https://open.er-api.com` when you open the currency-rates
screen. This request contains no personal data and no user identifiers — it
only retrieves public exchange-rate values.

## Permissions

- **Face ID / device biometrics** — used locally to lock the app, if you enable
  it. Authentication happens on-device; no biometric data is collected or
  transmitted.
- The app declares some system usage strings (camera, photo library, location)
  only because a bundled library references those APIs. Finance Bro never
  requests or uses those permissions, and no related data is collected.

## Children

The app does not knowingly collect data from anyone, including children.
